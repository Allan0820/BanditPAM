Swap step for the \mbox{\hyperlink{classFastPAM1}{Fast\+PAM1}} algorithm. 

Runs swap step for the \mbox{\hyperlink{classFastPAM1}{Fast\+PAM1}} algorithm. Loops over all datapoint and compute the loss change when a medoid is replaced by the datapoint. The loss change is stored in an array of size n\+\_\+medoids and the update is based on an if conditional outside of the loop. The best medoid is chosen according to the best loss change.
