Performs the BUILD step of \mbox{\hyperlink{classkm_1_1BanditPAM}{Bandit\+PAM}}. 

Draws batch size reference points with replacement and uses the estimated reward of adding candidate medoids to the set of medoids. Constructs confidence intervals for expected loss when adding each candidate as a medoid, and continues drawing reference points until the best candidate\textquotesingle{}s confidence interval is disjoint from all others.

Performs the SWAP step of \mbox{\hyperlink{classkm_1_1BanditPAM}{Bandit\+PAM}}. 

Draws batch size reference points with replacement and uses the estimated reward of performing a (medoid, non-\/medoid) swap. Constructs confidence intervals for expected loss when performing the swap, and continues drawing reference points until the best candidate\textquotesingle{}s confidence interval is disjoint from all others.
