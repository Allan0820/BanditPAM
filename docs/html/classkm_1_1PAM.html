Build step for the PAM algorithm.

Runs build step for the PAM algorithm. Loops over all datapoint and checks its distance from every other datapoint in the dataset, then checks if the total cost is less than that of the medoid (if a medoid exists yet).

Parameters
dataTransposed input data to cluster
medoid_indicesUninitialized array of medoids that is modified in place as medoids are identified
