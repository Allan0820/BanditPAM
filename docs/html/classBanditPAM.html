Build step for BanditPAM.

Runs build step for the BanditPAM algorithm. Draws batch sizes with replacement from reference set, and uses the estimated reward of the potential medoid solutions on the reference set to update the reward confidence intervals and accordingly narrow the solution set.

Parameters
dataTransposed input data to find the medoids of
medoid_indicesUninitialized array of medoids that is modified in place as medoids are identified
medoidsMatrix of possible medoids that is updated as the bandit learns which datapoints will be unlikely to be good candidates

Swap step for BanditPAM.

Runs Swap step for the BanditPAM algorithm. Draws batch sizes with replacement from reference set, and uses the estimated reward of the potential medoid solutions on the reference set to update the reward confidence intervals and accordingly narrow the solution set.

Parameters
dataTransposed input data to find the medoids of
medoid_indicesArray of medoid indices created from the build step that is modified in place as better medoids are identified
medoidsMatrix of possible medoids that is updated as the bandit learns which datapoints will be unlikely to be good candidates
assignmentsUninitialized array of indices corresponding to each datapoint assigned the index of the medoid it is closest to
